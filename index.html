CSS Transform

translate (x, y)

translate

translate - зсуває елемент на нове місце, переміщуючи щодо початкового положення об'єкта вправо та вниз, використовуючи координати X и Y. Записується так translate (x,y);

scale (x, y)

scale

scale - масштабує елементи, роблячи їх більше чи менше. За замовчуванням значення вказані як одиниця (1) – це реальний обсяг об'єкта. Перше значення масштабує елемент за ширині, друге - за висотою. Негативні значення відображають елемент дзеркальний.

rotate (вугол+deg)

rotate

rotate - повертає елементи на задане кількість градусів, негативні значення повертають елемент проти годинникової стрілки, позитивні - за годинниковою стрілкою. Записуються значення як градус повороту та deg

skew (x-вугол+deg, y-вугол+deg)

skew

skew - деформує (спотворює) сторони об'єкта щодо координатних осей. Якщо вказано одне значення, друге буде визначено браузером автоматично. Записуються значення як градус повороту та deg

matrix (a, c, b, d, x, y)

matrix

matrix(a, c, b, d, x, y) - дозволяє об'єднати кілька функцій 2D-трансформацій в одній. В якості трансформації допустимі поворот, масштабування, нахил (деформація) та зміна положення. Властивості вказуються без одиниць виміру.

Множинні 2D трансформації

translate
scale
rotate

Так само, подібного можна домогтися використовуючи множинні трансформації. А саме записати кілька значень в одне властивість перетворення.

transform-origin

transform origin

transform-origin - дозволяє змістити центр трансформації, щодо якої відбувається зміна положення/розміру/форми елемента

perspective, perspective-origin

perspective, perspective-origin

perspective, perspective-origin - встановлення глибини та зміна точки початку координат

translate3d (x, y, z)

translate3D

translate3D - за аналогією зі своїм 2D братом це значення задає переміщення елемента, проте, в 3D-просторі. Крім осі X та Y додається вісь Z.

scale3d (x, y, z)

scale3D

scale3D - задает трехмерное масштабирование по одной из осей X, Y, Z

rotate3d (x, y, z, deg)

rotate3d

rotate3d - вращает элемент относительно трех осей. Элемент поворачивается под углом, задаваемым последним параметром относительно вектора направления X, Y, Z

matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

matrix3d

matrix3d - задає тривимірне перетворення як однорідну матрицю розміром 4×4 з шістнадцятьма значеннями стовпцях

Множинні 3D трансформації

translate3d
rotate3d

Запис кількох 3D значень одна властивість transform.

transform
style

transform-style - задає стиль трансформації

Backface-visibility

FONT
BACK
